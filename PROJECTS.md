## How to Add a New Project

1. Create a new folder: `projects/project-name/`
2. Add these files:
   - `README.md` - Project overview
   - `test-plan.md` - Complete test plan
   - `test-cases/` - Individual test case files
   - `bug-reports/` - Bug documentation
   - `test-results.md` - Final results & metrics

3. Update this document with project details
4. Link to project folder in the table above
